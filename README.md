# OpenTrickler GUI Mockup

Interactive mockup of all three OpenTrickler user interfaces:

- **Web Portal** — phone-optimized web UI served by the Pico W / Pico 2W
- **TFT35** — 480×320 color touch display (RP2350 only), with rotation toggle
- **Mini 12864** — monochrome LCD with rotary encoder navigation (RP2040 & RP2350)

---

## 👉 [Click here to open the mockup](https://cavemanstoys.github.io/OpenTrickler-GUI-Mockup/)

---

Switch between the three interfaces using the tabs at the top of the page.

## Features Showcased

- 8-stage charge progress indicator (Zero → Coarse → Reverse → Fine → Pulse → Verify → Remove → Return)
- Shared top navbar with WiFi status, firmware version, theme toggle
- Charge simulation button
- Profile name under weight readout
- Coarse reverse with independent speed + revolutions (0 = disabled)
- Purge/cleanup mode
- Full settings pages (scale, profiles, charge mode, AI tuning, motors, WiFi, display, etc.)
- Flex sticky-footer layout — bottom nav never overlaps content
